# Use ScrollMagic & GSAP in Nuxt.js Demo

> 在 Nuxt.js 中使用 ScrollMagic 与 GSAP

## Build Setup

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
